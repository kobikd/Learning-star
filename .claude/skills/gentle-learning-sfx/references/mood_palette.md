# Mood & Texture Palette — כוכב הלמידה

Use these words in ElevenLabs prompts. Pick 2–4 per sound spec.
Hebrew terms are listed alongside English for reference when writing prompts.

## Warmth & softness (חמימות ורכות)
- soft / רך, warm / חמים, gentle / עדין, cozy / נעים, rounded / מעוגל,
  muted / מושתק, mellow / נינוח, velvety / קטיפתי, cushioned / מרופד

## Magic & wonder (קסם והפתעה טובה)
- magical / קסום, sparkly / נצנץ, twinkling / מנצנץ, shimmery / מבריק בעדינות,
  enchanted / מוקסם, fairy-like / כמו פיה, dreamy / חלומי, starry / כוכבי

## Nature (טבע)
- bubbly / בועתי, flowing / זורם, watery / מימי, dewy / טללי,
  breezy / אווירי, leafy / ירוק, earthy / אדמתי, wooden / עצי

## Friendliness (ידידותיות)
- friendly / ידידותי, cute / חמוד, cheerful / עליז, sweet / מתוק,
  playful / שובב, giggly / צוחקני, cuddly / חבקני, happy / שמח

## Calm & focus (שקט וריכוז)
- calming / מרגיע, soothing / מנחם, peaceful / שליו, serene / שקט,
  tranquil / רגוע, ambient / סביבתי, restful / נח

## Material textures — safe (מרקמים בטוחים)
- felt-like / כמו לבד, cottony / כותנתי, papery / נייר, clay-like / חרסינה,
  woolly / צמרי, fabric / בד

## Musical qualities — safe (איכויות מוזיקליות)
- melodic / מלודי, chime-like / כמו פעמון רך, marimba-like / כמו מרימבה,
  music-box-like / כמו קופסת נגינה, lullaby-like / כמו שיר ערש,
  pentatonic / פנטטוני, major-key / במינור גדול, ascending / עולה, resolving / מסיים

## Spatial (מרחב)
- airy / אוורירי, floating / צף, intimate / אינטימי, enveloping / עוטף

---

## Combining moods — examples (Hebrew app context)

- כפתור פרח נלחץ: "soft magical sparkly warm"
- תשובה נכונה: "warm cheerful ascending encouraging"
- רקע מסך שקט: "flowing peaceful misty calming"
- חתול מגיב: "cute giggly friendly warm"
- מעבר בין שלבים: "dreamy smooth floating gentle"
- שיר פתיחה: "magical playful lullaby-like warm"
